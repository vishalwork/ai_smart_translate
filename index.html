// GET STARTED

Install in seconds.

pubspec.yaml
dependencies: ai_smart_translate: ^1.1.0
main.dart
// 1. init once in main() await AiSmartTranslate.init( groqKey: 'YOUR_GROQ_KEY', // free · 1st priority geminiKey: 'YOUR_GEMINI_KEY', // free quota · 3rd priority ); // 2. wrap app runApp(TranslationScope(child: MyApp())); // 3. use .tr on any string — that's it Text('Hello'.tr) Text('Welcome, {name}'.tr) // placeholders preserved // 4. change language anytime await AiSmartTranslate.changeLanguage('hi');
// WHY USE THIS

Built different.

🧠
AI-Powered Natural Language

Goes beyond word-for-word translation. Uses AI to produce translations that actually sound natural in context.

🔄
Auto Fallback Chain

Groq → OpenRouter → Gemini → Google Translate. On rate limit, next provider kicks in automatically. Your app never breaks.

⚙️
Custom Provider Order

Define your own fallback chain with ProviderConfig. Override models, set priority, plug in any OpenAI-compatible endpoint.

🌐
All Platforms

Android, iOS, Web, Linux, Windows, macOS — all supported. Auto-selects the right cache backend per platform.

💾
Offline Cache

Translate once, use forever. SQLite on mobile/desktop, SharedPreferences on web. Zero repeated API calls.

🎯
Zero JSON Maintenance

No translation files to maintain. Add .tr to any string and the package handles the rest.
